# Welcome to react-api-git

React-api-git is a project with the objective of searching GitHub user profiles and repositories.
To be able to use the search features, you need to **authenticate** with the login from [**GitHub**](https://github.com/) itself

View their code in the `src/stories` directory to learn how they work.

<div className="subheading">Start</div>

To start the project it is necessary to clone the repository
[**react-api-git**](https://github.com/MatheusSan/react-api-git).

After cloning, run the `yarn install` command to install the project's
dependencies. Don't forget to install the backend dependencies `cd backend && yarn install`.

With the dependencies installed, run the project backend (required for
authentication) `cd backend && yarn start` and then start the project `cd .. && yarn start`.

<div className="subheading">Use</div>

With an authenticated GitHub user, there will be a screen displaying your data (photo,
name, nickname and bio). Right below you will have a search field where you can type
the GitHub username you want to search for.

With the field filled...

If you want to search by user only, press enter or click on the magnifying glass.

If you want to search for the repositories click on the button with the word 'repos'.

If you want to search for this user's favorite repositories click on the button with the word 'starred'.
